TUGAS 4#

Proyek Missing Values Inputation + Materi Normalisasi data

  1. Missing values : menyelesaikan dengan WKNN (manual) + code menghitung WKNN

  2. Normalisasi data (materi)

    • macam macam normalisasi data dan beri contoh hasil normalisasi data

    • Gunakan library sklearn untuk melakukan normalisasi data atau membuat fungsi sendiri

Perhitungan :

  • Preprocessing (macam-macam normalisasi data)

  • normalisasi merupakan tahapan preprocessing

  • pengerjaan missing value dengan WKNN

  1. PENGERTIAN MISSING VALUE

Missing value adalah kondisi dimana suatu atribut dalam dataset tidak memiliki nilai atau bernilai kosong. Hal ini sering terjadi karena kesalahan input, data tidak lengkap, atau kegagalan sistem saat pengumpulan data.

Dampak:

Mengurangi kualitas data

Menurunkan akurasi model

Menghambat proses analisis

Missing value dapat menyebabkan proses klasifikasi menjadi tidak optimal karena data tidak lengkap

  1. IMPUTASI MISSING VALUE

Imputasi adalah proses mengisi nilai yang hilang dengan nilai pengganti berdasarkan metode tertentu.

Tujuan:

  • Menghindari penghapusan data

  • Menjaga jumlah dataset tetap utuh

  • Meningkatkan performa model

Metode umum:

  • Mean / rata-rata

  • Median

  • Modus

  • KNN Imputation

  • WKNN (Weighted KNN)

Penggunaan metode imputasi terbukti dapat meningkatkan akurasi dibandingkan membiarkan data kosong

  1. KNN UNTUK MISSING VALUE

Metode K-Nearest Neighbor (KNN) digunakan untuk mengisi nilai kosong dengan melihat data terdekat (tetangga) berdasarkan kemiripan.

Konsep:

  • Hitung jarak antar data

  • Ambil K data terdekat

  • Gunakan nilai mereka sebagai acuan

KNN menggunakan jarak (biasanya Euclidean) untuk menentukan kemiripan antar data.

  1. WEIGHTED KNN (WKNN)

WKNN adalah pengembangan dari KNN, dimana setiap tetangga memiliki bobot berdasarkan jaraknya.

Konsep utama:

  • Data yang lebih dekat → pengaruh lebih besar

  • Data yang jauh → pengaruh lebih kecil

Rumus bobot:

\[ w_i = \frac{1}{d_i} \]

Rumus prediksi:​

\[ \hat{y} = \frac{\sum_{i=1}^{k} w_i \cdot y_i}{\sum_{i=1}^{k} w_i} \]

WKNN lebih akurat dibanding KNN biasa karena mempertimbangkan tingkat kedekatan data

  1. LANGKAH-LANGKAH WKNN

  • Melakukan preprocessing (normalisasi data)

  • Menentukan data yang memiliki missing value

  • Menghitung jarak ke semua data lain

  • Mengurutkan jarak (terkecil ke terbesar)

  • Mengambil K tetangga terdekat

  • Menghitung bobot tiap tetangga

  • Menghitung nilai imputasi menggunakan rumus WKNN

  • Mengisi nilai missing dengan hasil tersebut

  1. KELEBIHAN DAN KEKURANGAN WKNN

Kelebihan:

  • Lebih akurat dari KNN biasa

  • Mempertimbangkan kedekatan data

  • Cocok untuk imputasi missing value

Kekurangan:

  • Perhitungan lebih kompleks

  • Sensitif terhadap nilai K

  • Membutuhkan preprocessing (normalisasi)

1. PREPROCESSING DATA#

Preprocessing = tahap awal sebelum pemodelan

Tujuan:

  • Mengatasi data tidak lengkap (missing value)

  • Menyamakan skala data (normalisasi)

2. NORMALISASI DATA#

###A. Pengertian

Normalisasi : proses mengubah skala data agar berada dalam rentang tertentu (biasanya 0–1)

Normalisasi dilakukan agar semua fitur memiliki skala yang sama, sehingga tidak ada fitur yang mendominasi perhitungan jarak.

Tanpa normalisasi:

  • Data besar (contoh: 3.000.000) akan mendominasi

  • Hasil jarak menjadi tidak akurat

Dengan normalisasi:

  • Semua fitur berada dalam rentang yang sama

  • Perhitungan KNN/WKNN lebih adil

B. Macam-macam Normalisasi#

####1. Min-Max Normalization

Rumus:

\[ X' = \frac{X - X_{min}}{X_{max} - X_{min}} \]

  • Max = 4

C. Hitung Jarak (Euclidean)#

Rumus:

\[ d = \sqrt{(x_1 - x_2)^2 + (y_1 - y_2)^2} \]

atau $\( d = \sqrt{\sum_{i=1}^{n} (x_i - y_i)^2} \)$

Contoh ke data 1:

\[ d = \sqrt{(0 - 0)^2 + (0.5 - 0)^2} \]
# =========================
# 3. HITUNG JARAK
# =========================
train = df_norm.dropna()
test = df_norm[df_norm['JML'].isna()]

X_train = train[['IPK','PO']].values
y_train = train['JML'].values
X_test = test[['IPK','PO']].values[0]

# hitung jarak
distances = np.sqrt(np.sum((X_train - X_test)**2, axis=1))

df_jarak = train.copy()
df_jarak['Jarak'] = distances

E. Hitung WKNN (Weighted KNN)#

Rumus bobot:

\[ w_i = \frac{1}{d_i} \]

Hitung hasil:

\[ \hat{y} = \frac{\sum_{i=1}^{k} w_i \cdot y_i}{\sum_{i=1}^{k} w_i} \]
# =========================
# 5. BOBOT WKNN
# =========================
df_k['Bobot'] = 1 / df_k['Jarak']
